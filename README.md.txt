# Chat Persistente

Un chat simple que guarda todos los mensajes en el navegador (localStorage) y requiere nombre de usuario.

## Características

- Persistencia de mensajes (no se borran al recargar)
- Nombre de usuario personalizado
- Diseño responsive
- Fácil de implementar

## Cómo usar

1. Clona este repositorio
2. Sube los archivos a GitHub Pages
3. O simplemente abre el archivo `index.html` en tu navegador

## Personalización

Puedes modificar:
- Los colores en `style.css`
- El texto de las respuestas automáticas en `script.js`
- El título de la página en `index.html`

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage para persistencia
